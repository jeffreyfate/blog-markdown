Have you imported a Gradle v5.0+ project into Intellij and been greeted with an error with the first build? Something like this:
```
java.lang.AbstractMethodError: org.jetbrains.plugins.gradle.tooling.util.ModuleComponentIdentifierImpl.getModuleIdentifier()Lorg/gradle/api/artifacts/ModuleIdentifier;
```
If you are running Intellij 2018.1, you'll need to upgrade to 2018.2 or better to resolve this issue. I upgraded to 2019.1.3 and that resolved the problem.
